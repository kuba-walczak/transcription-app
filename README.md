# Live Transcription for vMix & YouTube

Desktop application built for a YouTube livestream of the [Institute for Educational Research](https://ibe.edu.pl).  
The app acts as a middleman between vMix, YouTube, and Speechmatics APIs to deliver real-time captions.

## ✨ Features

- Real-time speech transcription from host audio input
- Audio routed to Speechmatics API and converted into caption-ready output
- Integration with YouTube and vMix APIs for live broadcast captions
- Java Swing desktop GUI for simple and reliable control during streams

## 🛠 Tech Stack

- Java
- Swing
- Speechmatics API
- YouTube API
- vMix API
